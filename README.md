# ArknightsIDCard
明日方舟长名片制作器

使用说明：  
依照source里的excel编辑自己的练度，注意干员名不能有错和表有两页  
需要在new_draw.py的同级目录新建一个OutPut文件夹用于输出  
然后运行new_draw.py或者new_draw.exe
